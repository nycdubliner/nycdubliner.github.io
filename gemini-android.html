Gemini CLI on Android

Setup Instructions

To get Gemini CLI running on Android, you'll need Termux. Follow these steps:

  1. Install Termux (preferably from F-Droid).
  2. Update packages: pkg update && pkg upgrade
  3. Install core dependencies: pkg install nodejs python git gh
  4. Install Gemini CLI: npm install -g @google/gemini-cli

Enhancing the Experience

Android Notifications: You can send alerts to your phone from Termux by installing the termux-api package (and the corresponding Termux:API app from your app store):

pkg install termux-api
termux-notification -t "Hello" -c "Gemini task complete!"

Typing Shortcut: For a more natural typing experience in Termux, you can toggle regular Android keyboard features (like swipe-to-type or autocomplete) by swiping the special keys bar above the keyboard.

Dev Tools: This environment gives you full access to git and the GitHub CLI (gh), allowing for a professional mobile development workflow.
